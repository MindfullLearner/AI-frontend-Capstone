# Internship Learning Journal

## Week 1 – Environment Setup

During the first week of the FlyRank AI Frontend Engineering Internship, I learned how to:

- Install and configure Node.js and Git.
- Create and organize a professional GitHub repository.
- Write a structured README.md file.
- Create a CLAUDE.md file with project guidelines.
- Understand and use Conventional Commits.
- Set up AI-assisted development tools.
- Use Cursor AI to review and improve project documentation.
- Work with Git commands such as:
  - git add
  - git commit
  - git pull
  - git push
- Resolve a Git synchronization issue between my local repository and GitHub.

## Key Takeaway

A clean project structure, meaningful commit history, and AI-assisted development are important parts of a professional software engineering workflow.
